What is 'AIO'?

AIO stands for 'ALL IN ONE'. As it states AIO is a revolutionary concept and now its reality. It is a floating button on your website! Hovering your mouse over it reveals a menu with many options.


  • ScreenPDF
  • The first option is ScreenPDF. As the name suggests, clicking this option will create a .pdf file of the current page and will start automatic download. SCREEN PDF USES AN EXTERNAL API SO IT IS DEPENDENT ON THE API WHICH IS DEPENDENT ON THE RESPECTIVE WEBSITE.

  • ScreenShot
  • Always wanted to take a screenshot of the webpage as we do on our phones? Now you can! clicking 'ScreenShot' will generate a screenshot of the current page and will start automatic download. The screenshot will be saved as "download", you can open the file using any application like MS Word, Paint (Recommended).

  • Print
  • Print the screen easily just by one click. Not supported on mobile devices.

  • Full Screen
  • Don't let other running disturb you. Go fullscreen and concentrate of the webpage you like.
